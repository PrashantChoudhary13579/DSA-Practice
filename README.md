# DSA-Practice

### ADD
``` git add filename.cpp ``` 
or 
```git add .```

### Undo
```git restore --staged filename``` or ```git restore --staged .```

### Commit 
```git commit -m "comment" ```

### Push
- if doing it for the first time and what to save the path

    ```git push -u origin main ```

or 
- for generally 

    ```git push ```
